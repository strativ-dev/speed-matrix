== Description ==

Speed Matrix is a comprehensive WordPress performance optimization plugin designed to help you achieve high PageSpeed scores and improve overall site speed.

**Key Features:**
* Page Caching with static HTML generation
* CSS Minification & Combining
* JavaScript Minification & Defer
* Image Lazy Loading
* WebP Image Support
* Browser Caching & GZIP Compression
* Performance Tweaks (Remove Emojis, Embeds, Query Strings)
* Export/Import Plugin Settings

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/speed-matrix/`
2. Activate the plugin from the Plugins menu in WordPress
3. Go to Settings > Speed Matrix
4. Choose a preset (Recommended for best results)
5. Enable the desired optimizations
6. Save changes and test your website

== Frequently Asked Questions ==

= Will this plugin break my website? =
Start with the "Recommended" preset and test your site. If issues occur, disable specific options individually.

= Do I need to clear cache after updates? =
Yes, always clear your cache after theme, plugin, or WordPress updates.
